### bijiao A B
A and B can be folder path or file list

example
- bijiao A.txt B.txt
- bijiao A.txt /home/nihui/B
- bijiao /home/nihui/A /home/nihui/B

output
- A-list.txt
- B-list.txt
- A-B-dou-you.txt
- A-you-B-meiyou.txt
- B-you-A-meiyou.txt
